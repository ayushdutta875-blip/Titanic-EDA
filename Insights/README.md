#  Titanic Survival Analysis

## Project Overview

This project is a simple Exploratory Data Analysis (EDA) of the Titanic dataset. The goal is to understand which factors influenced passenger survival during the Titanic disaster.

The analysis focuses on data cleaning, exploration, and visualization to identify patterns related to age, gender, passenger class, and family size.

## 📊 What I Analyzed

- Survival distribution of passengers
- Impact of gender on survival
- Effect of passenger class (Pclass) on survival
- Relationship between age and survival
- Influence of family size and traveling alone
- Correlation between numerical features

## 🛠 Tools Used

- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## 📈 Key Insights

- Women had a significantly higher survival rate than men.
- First-class passengers were more likely to survive than third-class passengers.
- Children had better chances of survival compared to adults.
- Passengers traveling alone showed different survival patterns compared to those traveling with family.
- Gender and passenger class were the strongest factors influencing survival.
